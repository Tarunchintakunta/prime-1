# Primelearning

Dark, premium landing page for Primelearning — an online learning platform.

Built with React + Vite + TypeScript + Tailwind CSS + Framer Motion + lucide-react + shadcn/ui (Accordion).

## Develop

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy

Push to GitHub and import in Vercel. No env vars needed. `vercel.json` provides SPA fallback.
